# VocabLingo Support

## About
VocabLingo is a bilingual (German/English) vocabulary learning app for iPhone, iPad, and Mac.

## FAQ

### How do I start a quiz?
Tap "Quiz starten" on the home screen, then select the question count and mode.

### What's the difference between EN→DE and DE→EN?
- **EN→DE:** See an English word → choose the correct German translation
- **DE→EN:** See a German word → choose the correct English translation

### When is a word considered "mastered"?
After one correct answer (correctCount >= 1).

### Why can't I see all 3014 words?
Until you reach 25% progress, only A1 beginner words (887) are shown. All 3014 words unlock at 25%.

### How does error review work?
Go to Error Review to practice only the words you answered incorrectly. Answer correctly twice (over multiple sessions) to clear the error.

### Can I sync progress between devices?
No – progress is stored locally on each device and cannot be synced.

### How do I reset all progress?
Go to Settings → "Alles zurücksetzen" (Reset All).

## Privacy Policy
VocabLingo does **not** collect, store, or transmit any personal data. All app data (vocabulary progress, favorites, settings) is stored locally on your device using UserDefaults. No internet connection is required.
